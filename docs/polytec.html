Polytec vibrometer module¶

The Polytech vibrometer instrument.

This module allows interfacing with Polytec vibrometer OFV-5000 controller and OFV-505 sensor head. Functions based on Polytec “RS-232 Interface Commands: OFV-5000 User Manual”

NOTE For each polytec controller, different decoders may be installed. These values should be stored in the your PLACE config file. (~/.place.cfg)

Example: DD-300 is ‘DisplDec,0’ DD-900 is ‘DisplDec,1’ VD-08 is ‘VeloDec,0’ VD-09 is ‘VeloDec,1’
