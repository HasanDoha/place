
=====
PyPAL  
=====

PyPAL is an open-source Python package for laboratory automation and data analysis.  
PyPAL includes Python modules for automating instruments used for non-contact laser-generated ultrasound experiments, example implementation scripts, and modules for plotting and processing data.
