
=====
PyPAL  
=====

PyPAL is an open-source Python package for laboratory automation and waveform analysis.  
PyPAL includes Python modules for automating instruments used for non-contact laser-generated ultrasound experiments, example implementation scripts, and modules for visualizing and processing waveform data.
